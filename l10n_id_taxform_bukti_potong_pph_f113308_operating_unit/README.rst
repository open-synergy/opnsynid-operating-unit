========================================================================
Indonesia - Bukti Potong PPh Pasal 26 (f.1.1.33.08) with Operating Units
========================================================================

Adds the Operating Unit in Bukti Potong PPh Pasal 26 (f.1.1.33.08).

Installation
============

To install this module, you need to:

1.  Clone the branch 8.0 of the repository https://github.com/open-synergy/opnsynid-operating-unit
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Setting -> Modules -> Local Modules*
5.  Search For *Indonesia - Bukti Potong PPh Pasal 26 (f.1.1.33.08) with Operating Units*
6.  Install the module
